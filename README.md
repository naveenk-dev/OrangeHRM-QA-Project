# orangehrm qa testing project

this repository contains manual testing documents and selenium automation framework for the orangehrm demo application.

application url
https://opensource-demo.orangehrmlive.com/

---

# project overview

this project shows a simple qa testing workflow from starting to ending.

things covered in this project

* requirement analysis
* test planning
* test scenario writing
* test case writing
* bug reporting
* rtm (requirement traceability matrix)
* test summary report
* selenium automation framework

---
