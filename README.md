# 📌 Project Overview

This repository demonstrates a complete **Software Quality Assurance (QA)** workflow for the **OrangeHRM Demo Application**, covering both **Manual Testing** and **Selenium Automation Testing** using industry-standard tools and best practices.

The objective of this project is to simulate a real-world QA environment by performing complete software testing activities—from **Requirement Analysis** and **Test Planning** to **Automation Framework Development** and **Test Execution**.

# ▶️ Getting Started

## Clone Repository

```bash
git clone https://github.com/naveenk-dev/OrangeHRM-QA-Framework.git
```

---

## Navigate to Project

```bash
cd OrangeHRM-QA-Framework
```

---

## Install Dependencies

```bash
mvn clean install
```

---

## Execute Tests

```bash
mvn test
```

or execute

```text
TestNG.xml
```

using Eclipse.
